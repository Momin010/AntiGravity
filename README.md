# macOS Web Simulation

This is a React + Vite project simulating the macOS interface.

## Prerequisites

You need to have Node.js and npm installed on your machine.

## Setup & Run

1.  Open a terminal in this directory.
2.  Install dependencies:
    ```bash
    npm install
    ```
3.  Start the development server:
    ```bash
    npm run dev
    ```
4.  Open your browser to the URL shown (usually `http://localhost:5173`).

## Features

-   **Lock Screen**: Enter any password to unlock.
-   **Desktop**: Realistic wallpaper and menu bar.
-   **Dock**: Magnification effect on hover.
-   **Windows**: Draggable and closable windows.
-   **Apps**:
    -   **Finder**: Basic file explorer layout.
    -   **Safari/Messages/etc**: Placeholder icons in the dock.

## Notes

-   The project uses **Framer Motion** for animations.
-   Styles are written in **SCSS**.
